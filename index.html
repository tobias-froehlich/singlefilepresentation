How to make
slides for a presentation
using
a single HTML file
Tobias Fröhlich
Text
This template consists of a single HTML file. The styling of the slides is done via CSS. Some simple adjustments can be done without knowlage of HTML and CSS. For example the colors can be found in variables that can be easily changed.
It is possible to define CSS classes that center-align a text.
Or a CSS class that aligns the text to the right.
It is also possible
to use position: absolute
for floating text boxes.
Alternatively with a border.
The footer with the slide number is automatically added using JavaScript.
Tables and Lists
Tables can be done as usually in HTML and they can be styled e.g. like this:
First ColumnSecond Column

very, very, veeery loooong entryfour
The same for unordered lists:
  • Apple
  • Banana
  • Grapefruit
Diagrams
Diagrams can also be build using normal HTML. This has the advantage that the styling of the font and the borders is consistent with the rest of the presentation.
Only the connection lines like arrows are a bit tricky. They are done via a placeholder element.
<div class="line">{"from": "bla.SE", "to": "blubb.NW", "style": "bidirectionalArrow"}</div>
This placeholder element is then parsed by JavaScript and an arrow is rendered using SVG. The connected elements must have an id. For example, "bla.SE" means that the South-East corner of the Element "bla" is used.

Workflow
Start with this template
Open the file a Browser
Edit the file with a text editor while watching the changes in the browser
Use the print functionality of the browser in order to generate a PDF file
{"from": "step-1.E", "to": "step-2.W", "style": "normalArrow"}
{"from": "step-2.E", "to": "step-3.W", "style": "normalArrow"}
{"from": "step-3.E", "to": "step-4.W", "style": "normalArrow"}
(You can also place images next to the HTML file and use them like usually in HTML. But then it is not single-file any more.)
